# Shopee Review Crawler Extension

## Cài đặt Extension

### Bước 1: Tạo Icon
Tải icon 16x16, 48x48, 128x128 pixels và đặt vào thư mục này với tên:
- icon16.png
- icon48.png  
- icon128.png

Hoặc dùng icon mặc định bằng cách download từ: https://icons8.com/icons/set/shopping-cart

### Bước 2: Cài Extension vào Edge/Chrome

1. Mở Edge/Chrome
2. Vào `edge://extensions` hoặc `chrome://extensions`
3. Bật **Developer mode** (góc phải trên)
4. Click **Load unpacked**
5. Chọn thư mục `extension` này

### Bước 3: Sử dụng

1. Mở trang sản phẩm Shopee bất kỳ
2. Click icon extension trên toolbar
3. Nhấn nút **🚀 Crawl Tất Cả Reviews**
4. Đợi và lưu file JSON

## Features
- ✅ Tự động detect Shop ID và Item ID
- ✅ Hiển thị tổng số reviews
- ✅ Crawl TẤT CẢ reviews (không giới hạn)
- ✅ Download JSON tự động
- ✅ Không bị chặn bởi anti-bot (chạy trong browser context)
